ANSWER .2 Install the latest version of Microsoft DirectX. Chances are, upgrading to the latest version will fix the error "d3dcompiler_43.dll not found," but if not, try DirectX End-User Runtimes (June 2010) instead.

ANSWER .3 Assuming the latest DirectX version from Microsoft doesn't fix the error you're receiving, look for a DirectX program on your game or application DVD or CD. Sometimes, though not often, the version included on the disc is a better fit for the program than the latest version available online.
